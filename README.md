## Job Market Analysis Dashboard

## Project Overview
Analyzed 61,953 job postings to identify salary trends, remote work opportunities, job roles, and in-demand skills.

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI

## Key Insights
- Average Salary: $92,289
- Total Jobs Analyzed: 61,953
- Top Role: Data Analyst
- Most Demanded Skills: SQL, Excel, Python, Power BI, Tableau

## Dashboard Features
- Salary Analysis
- Job Role Analysis
- Remote Work Analysis
- Skill Demand Analysis
